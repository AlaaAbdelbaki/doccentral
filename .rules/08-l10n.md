# 🌍 Localization Rules (L10n)

## 🧠 Supported Languages

* English (default)
* French
* Arabic (RTL)

---

## 🚨 Rules

* No hardcoded strings in UI
* All text must use localization keys
* No business logic in localization layer

---

## 📦 Structure

All translations must be stored in ARB files:

* app_en.arb
* app_fr.arb
* app_ar.arb

---

## 🧭 RTL Support

* Arabic must support RTL layout automatically
* No manual RTL handling in UI widgets

---

## 🧠 Golden Rule

UI MUST NEVER contain raw strings
